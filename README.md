# Helium mod
Client-side performance optimizations for Minecraft. Faster math, less GC pressure, smarter rendering. Zero visual changes. Requires Sodium. 

[NOTE] This mod is not made by me, it was a super powerful mod which desappear from the internet and i just archived it and wont be working on it, the .jar file is decopiled using CFR 0.152. The original .jar file is avalable in package.
